This lab's main.js file is a reference for potentially useful string sanitization on text you will query from Huggingface.

The cleanUpText() function in main.js contains some common text cleanup operations, specifically geared toward text that was parced by OCR.

Play around with it, modify it, make it fit your needs.


The main upside to using some simple javascript like this instead of calling a library or LLM to fix your text, is that it is fast and free.

The main downside is that it might not as powerful and precise at correcting as an LLM.


If you want to experiment with LLMs I suggest reading this guide by OpenAI on:
  1. how to make an api key
  2. how to add it to an environment variable
  3. how to call the api

As you'll notice in the guide, this will require you to have a more complete web application set up, with npm (node package manager) so you can install the OpenAI library.
